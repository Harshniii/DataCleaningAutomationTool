The Data Cleaning Automation Tool is a Python-based web application built with Streamlit that automates essential data preprocessing tasks. It allows users to upload a dataset (CSV) and automatically performs key cleaning operations like:

Handling missing values (by filling or removing them)

Detecting and removing outliers

Fixing data inconsistencies (like capitalization or whitespace errors)

Providing a summary report after cleaning

This tool is useful for data analysts, data scientists, and students who want to quickly clean and preprocess datasets before analysis or machine learning tasks.
